Technologies Used in My Solution

Here I will hightlight the technologies I used to implement my solution to this challenge. Those technologies include: HTML, Javascript, Bootstrap, Github (for code management), and Netlify (for solution deployement). I will get into some details below.

Key Skills
  • JavaScript Modulus Operator - If I had point to one piece of knowledge that this coding challenge required, it would be the use of the modulus operator in JavaScript. The primary objective of this challenge is to execute special handling of the output when numbers between 1 and 100 can be evenly divided into the numbers entered by the user. The modulus operator will return the remainder of a division operation. When the remainder is equal to zero, the numbers are evenly divided. That is the test we must perform before we can know whether or not the number deserves special handling.
  • HTML Templating - I chose to offer two options to the user for how they view the results. I used the HTML template tag to accomplish this objective. The HTML template tag allows me to write HTML code that is not rendered on the page, but can be cloned in JavaScript in order to build a output display the consistent with the user's objective. The key is that we don't know the usre's objective until runtime, so we have to wait until then to choose the correct display. I created templates for both the table output and the text block output, and chose which one to use based on user input.
  • Array Processing - The rules require that we test numbers from 1 to 100. An array seems to be a natural data structure for this task. However, in the end, the results needed to be converted to an HTML output. So there are two basic steps: 1. build the array, and 2. build the HTML output based on the array contents.
  • Nested Loops - I chose to offer the user the option to see the output in a table format. Since there are 100 numbers to test, that meant there are 100 separate items that needed to be displayed. The perfect structure for the table therefore, would be table with 10 rows, each with 10 columns. So while looping through the output array, I had to be aware of which cell in the table I was working on. So I created a loop for each row, nested inside a loop for each column.
